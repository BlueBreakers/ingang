# 인강호흡
> 인터넷 강의 수강 및 교재 판매 사이트 제작


## 프로젝트 설명
인터넷 강의를 수강하고 교재를 구매하여 학습 할 수 있는 사이트 입니다. 


## 주요 기능
- 강의를 수강신청하고 수강평을 등록할 수 있다.


## 기술스택
`JavaScript`,`HTML`,`CSS`,`Java`,`Spring`,`OracleDB`
